1.  Open Antigravity → Manager View. Feed this TTB and the TIP (Technical Implementation Plan) 
as context files.  
2.  Feed ONE Epic at a time. Never all 8 at once. After each CHECKPOINT, open a fresh agent 
session for the next Epic.  
3.  Agent colour coding  →  
BLUE task cards  → Gemini 3 Pro in Manager View. Boilerplate, scaffolding, UI. Uses free -tier 
tokens.  
PURPLE task cards  → Claude Sonnet 4.6. Math -critical, security, complex logic. Reserve Claude 
credits for these 8 tasks only.  
4.  CHECKPOINT blocks are hard stops — do not proceed to the next Epic until all gate checks 
pass. Prevents cascading failures and wasted tokens.  
5.  PARALLEL: YES tasks can be spawned as concurrent agents in Manager View. Run them at the 
same time as their listed companion task.  
6.  Appendix A = agent routing quick -ref.  Appendix B = GitHub Issues push script.  Appendix C = 
Frontend Design Skill for this project.

All in Books should feel like a premium physical ledger reimagined as a native Android app. The aesthetic 
is refined and utilitarian — not playful, not generic SaaS blue -white, not material design defaults. Think: 
quality stationery, clean ink on cream pap er, confident numbers. The single unforgettable thing: every 
number is perfectly formatted and colour -coded green/red with absolute precision.
